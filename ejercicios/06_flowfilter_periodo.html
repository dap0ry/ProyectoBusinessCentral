EJERCICIO 06

FlowFilters: Ventas por Período

Dificultad: 7/10
Objetivo: Crear un FlowFilter que permita filtrar los cálculos de un FlowField por un rango de fechas.

Instrucciones

  • Crea un campo "Date Filter" en tu tabla de estadísticas.
  • Configura FieldClass = FlowFilter; y tipo Date.
  • Modifica un FlowField existente (como "Total Ventas") añadiendo a su CalcFormula una condición: "Posting Date" = FIELD("Date Filter").
  • Prueba a filtrar la página por diferentes fechas para ver cómo cambia el valor.
field(100; "Date Filter"; Date)
{
    FieldClass = FlowFilter;
    Caption = 'Filtro Fecha';
}

field(101; "Ventas Periodo"; Decimal)
{
    FieldClass = FlowField;
    CalcFormula = Sum("Sales Line".Amount WHERE ("Posting Date" = FIELD("Date Filter")));
}
