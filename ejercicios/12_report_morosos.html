EJERCICIO 12

Reports: Listado de Morosos

Dificultad: 8/10
Objetivo: Crear un informe profesional que liste los socios con préstamos atrasados, utilizando un dataset jerárquico.

Instrucciones

  • Define un objeto Report llamado "Overdue Loans".
  • Añade un DataItem principal para la tabla Customer.
  • Anida un segundo DataItem para la tabla "Book Loan".
  • Añade una RequestPage donde el usuario pueda seleccionar una "Fecha de Corte" para considerar el retraso.
  • Exporta las columnas de nombre del socio y título del libro a la plantilla.
dataset
{
    dataitem(Customer; Customer)
    {
        column(CustomerName; Name) { }
        dataitem(BookLoan; "Book Loan")
        {
            DataItemLink = "Customer No." = field("No.");
            column(BookTitle; "Book Title") { }
        }
    }
}
