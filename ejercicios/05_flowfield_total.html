EJERCICIO 05

FlowFields: Total Prestado

Dificultad: 6/10
Objetivo: Implementar un campo de tipo FlowField que cuente automáticamente cuántos libros tiene prestados un socio.

Instrucciones

  • Extiende la tabla de Customer o usa una tabla propia.
  • Crea un campo llamado "Active Loans" de tipo Integer.
  • Configura FieldClass = FlowField;.
  • Define la CalcFormula para que cuente (Count) los registros de la tabla "Book Loan" donde el campo "Customer No." coincida con el número del cliente actual.

Ayuda de AL

field(50100; "Active Loans"; Integer)
{
    FieldClass = FlowField;
    CalcFormula = Count("Book Loan" WHERE ("Customer No." = FIELD("No.")));
    Editable = false;
}
