EJERCICIO 11

Queries: Top 5 Autores

Dificultad: 7/10
Objetivo: Crear una Query de alto rendimiento que agrupe los libros por autor y sume el total de páginas escritas por cada uno.

Instrucciones

  • Crea un objeto Query llamado "Top Authors".
  • El DataItem principal debe ser la tabla Book.
  • Añade la columna Author y configúrala para agrupar (Group By).
  • Añade la columna "No. of Pages" y usa el método Sum para totalizar.
  • Configura el TopNumberOfRows = 5 para obtener el ranking.
query 50100 "Top Authors"
{
    QueryType = Normal;
    TopNumberOfRows = 5;
    
    elements
    {
        dataitem(Book; Book)
        {
            column(Author; Author) { }
            column(TotalPages; "No. of Pages")
            {
                Method = Sum;
            }
        }
    }
}
