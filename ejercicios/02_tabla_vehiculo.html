EJERCICIO 02

Tablas: Ficha de Vehículo

Dificultad: 3/10
Objetivo: Crear una tabla técnica para gestionar una flota de vehículos, utilizando diversos tipos de datos de AL.

Instrucciones

  • Crea la tabla Vehicle con ID 50101.
  • Debes incluir los siguientes campos:
    • Matrícula: Code[20]
    • Modelo: Text[100]
    • Fecha de Matrículación: Date
    • Kilometraje: Decimal
    • Es Eléctrico: Boolean

Esquema de la Tabla

table 50101 Vehicle
{
    DataClassification = ToBeClassified;
    
    fields
    {
        field(1; "Plate No."; Code[20]) { Caption = 'Matrícula'; }
        field(2; Description; Text[100]) { Caption = 'Modelo'; }
        // Implementa los campos restantes...
    }
}
