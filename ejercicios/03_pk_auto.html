EJERCICIO 03

PK: Numeración Automática

Dificultad: 4/10
Objetivo: Crear una tabla de histórico de eventos donde la clave primaria sea un número entero que crezca automáticamente.

Instrucciones

  • Crea la tabla History Log con ID 50102.
  • Define un campo Entry No. de tipo Integer.
  • Configura la propiedad AutoIncrement = true; en dicho campo.
  • Asegúrate de que Entry No. sea la Clave Primaria en la sección keys.
  • Añade un campo Description de tipo Text[250].

Ayuda

table 50102 "History Log"
{
    fields
    {
        field(1; "Entry No."; Integer)
        {
            AutoIncrement = true;
        }
        // ... resto de campos
    }
    
    keys
    {
        key(PK; "Entry No.") { Clustered = true; }
    }
}
