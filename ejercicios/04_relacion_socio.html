EJERCICIO 04

Relations: Socio y Préstamo

Dificultad: 5/10
Objetivo: Vincular una tabla de préstamos con la tabla maestra de clientes (Socios) de Business Central.

Instrucciones

  • Crea la tabla Book Loan con ID 50103.
  • Crea un campo Customer No. (Code[20]).
  • Añade la propiedad TableRelation para que este campo solo permita elegir valores que existan en la tabla Customer.
  • Recuerda que en la tabla Customer, el campo clave es "No.".

Fragmento de AL

field(2; "Customer No."; Code[20])
{
    // Vincula este campo con la tabla Customer
    TableRelation = Customer."No.";
}
