EJERCICIO 01

Enum: Estados de Alquiler

Dificultad: 2/10
Objetivo: Crear un objeto de tipo Enum en AL que gestione los posibles estados de una maquinaria en una empresa de alquiler.

Instrucciones

  • Crea un archivo llamado RentalStatus.Enum.al.
  • El Enum debe tener el ID 50100.
  • Debe contener los siguientes 4 valores:
    0: Disponible, 1: Alquilado, 2: En Reparación, 3: Retirado.
  • Asegúrate de que sea Extensible = true.

Pista de Código

enum 50100 "Rental Status"
{
    Extensible = true;
    
    value(0; Available) { Caption = 'Disponible'; }
    // Añade el resto de valores aquí...
}
