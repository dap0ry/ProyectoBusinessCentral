Proceso de Instalación

Sigue los pasos cronológicamente para configurar tu entorno de desarrollo.

1. Instalación de Docker

Para usar el instalador de Business Central, primero necesitas Docker Desktop. Asegúrate de:

  • Habilitar WSL 2 en Windows 11.
  • Configurar contenedores de Windows si es necesario (o Linux según la imagen).
  • Tener al menos 8GB de RAM dedicados.

2. Configuración de VS Code

Instala las herramientas necesarias para desarrollar en AL:

  • Descarga e instala Visual Studio Code.
  • Configura el archivo launch.json para conectar con tu contenedor de Docker.

3. Despliegue del Contenedor

Usa scripts de PowerShell o el asistente de BC para levantar la imagen:

  • Descarga la imagen oficial de Microsoft Artifact Registry.
  • Acepta el EULA y configura las credenciales de administrador.
  • Verifica el acceso vía Web Client antes de empezar a programar.

4. Extensión AL

El núcleo del desarrollo en Business Central:

  • Instala la extensión "AL Language" desde el marketplace.
  • Crea un nuevo proyecto con `AL: Go!`.
  • Descarga los símbolos del servidor para habilitar el IntelliSense.
